What is RoundPenny?

A round-up micro-transaction platform

Every time a user makes a purchase, RoundPenny automatically rounds up the transaction
to the nearest dollar and invests the spare change into a diversified portfolio.

💰 Automatic Round-Ups
Seamless spare change capture
📈 Micro-Investing

Complete API Flow

1POST /v1/auth/register — Create user account
2POST /v1/auth/login — Get JWT access token
3POST /v1/merchants — Onboard a merchant
4POST /v1/payments — Create $10.00 payment
5POST /v1/payments/{id}/confirm — Confirm → transaction_id
6POST /v1/webhooks — Register callback URL
7POST /v1/analytics/events — Track user event
8POST /v1/fraud/rules — Create detection rule
9GET /v1/auth/me — Fetch user profile
10POST /v1/auth/refresh — Rotate JWT token
11POST /v1/auth/logout — Sign out (HTTP 204)
